This folder contains implementation and usage documentation for postcanvas.

## Contents

- [Getting Started](getting-started.md)
- [Configuration Reference](config-reference.md)

- How to create single-image posts and carousels
- Model-level configuration for backgrounds, text, images, shapes, tables, and charts

- Shared font resolution behavior and fallback strategy
- Practical examples for local generation and cloud-friendly workflows

## Suggested Reading Order

1. Start with [Getting Started](getting-started.md)
2. Use [Configuration Reference](config-reference.md) while building layouts
3. Read [Renderer Architecture](renderer-architecture.md) when customizing internals
4. Run scripts from [Examples Guide](examples.md)
